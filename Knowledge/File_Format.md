### File_Format 檔案格式 ###

Json

通用 , 跨語言

    {   
        "user":
        {
            "name":"heylin",
            "age":29
        }
    }

XML

結構保存較完整

    <?xml version="1.0"?>
    <user>
        <name>heylin</name>
        <age>29</age>
    </user>



YAML

可讀性高 , 表達資料序列化的格式

    ---
        name:   heylin
        age:    29
    ---
